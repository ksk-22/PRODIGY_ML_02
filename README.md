# PRODIGY ML Task 02 - Customer Segmentation using K-Means

## Objective
Create a K-Means clustering model to group retail store customers based on their purchase history.

## Dataset
Mall Customers Dataset

## Libraries
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Steps
1. Load dataset
2. Data preprocessing
3. Find optimal clusters using Elbow Method
4. Train K-Means model
5. Visualize customer segments

## Output
Customer clusters and segmentation visualization.
